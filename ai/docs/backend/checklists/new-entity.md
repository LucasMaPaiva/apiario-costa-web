# Checklist New Entity

- existe migration para a nova estrutura?
- nomes de tabela e colunas seguem padrao consistente?
- foreign keys e cascades foram definidas corretamente?
- existe model para a entidade?
- o model declara relacionamentos?
- o model declara casts necessarios?
- existe repository para acesso a dados?
- existe service para regra de negocio?
- existe seeder se a entidade depender de dados base?
- existe resource se a entidade for exposta na API?
- existe request para entradas da entidade?
- existem testes de criacao, leitura e regra critica?
