# Checklist New Feature

- a feature esta no modulo correto?
- a rota foi criada ou localizada?
- existe DTO quando o envio exige transformacao?
- existe service para API?
- existe hook para orquestracao?
- existe validator se houver formulario?
- o componente ficou focado em exibicao?
- a pagina integrou a feature sem absorver logica demais?
- os nomes seguem o padrao?
